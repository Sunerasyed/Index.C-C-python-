# 🤖 AI Chatbox

A modern AI chat application inspired by ChatGPT with memory, voice support, and a clean UI.

## 🚀 Features
- 💬 Real-time AI chat  
- 🧠 Chat history & memory  
- 🎤 Voice input & output  
- 🔐 Secure backend API  
- 📱 Responsive designing system 

## 🛠️ Tech Stack
- Frontend: React.js, Tailwind CSS  
- Backend: Python (Flask/FastAPI)  
- Database: SQLite / MongoDB
